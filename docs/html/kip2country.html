NAME

kip2country - Lookup Country by IP Address.  

SYNTAX

kip2country ip [ip2] [ip3] ...  

DESCRIPTION

Lookup Country by IP Address.  

EXAMPLES

$ kip2country www.honeypot.co.uk

DE, Germany

$ kip2country 66.35.250.150

US, United States
